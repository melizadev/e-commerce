# E-Commerce

Aplicación de comercio electrónico desarrollada con React, Node.js, Express y MongoDB.

## Tecnologías

### Frontend

- React
- Redux Toolkit
- Tailwind CSS
- React Router
- i18next

### Backend

- Node.js
- Express
- MongoDB
- JWT Authentication

## Arquitectura de despliegue

Frontend:

- Vercel

Backend:

- Render

Base de datos:

- MongoDB Atlas

## Variables de entorno

### Frontend

Crear un archivo `.env`:

```env
VITE_API_URL=
```

### Backend

Crear un archivo `.env`:

```env
MONGO_DB_URL=
MONGO_DB_USERNAME=
MONGO_DB_PASSWORD=
MONGO_DB_NAME=
FRONTEND_URL=
PORT=
JWT_SECRET_KEY=
```

## Instalación

### Backend

```bash
npm install
npm run dev
```

### Frontend

```bash
npm install
npm run dev
```
